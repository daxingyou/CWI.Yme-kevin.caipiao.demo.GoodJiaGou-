1
系统设置
1.1 登录

1.1.1 关注并进入“微云打”微信服务号。

1.1.2 点击微信公众号关注回复或者下方菜单栏中“我的门店”-“系统设置”选项，进入“系统设置”页面。

1.1.3 点击“登录”按钮，输入账号绑定的手机号码，点击右方“获取验证码”按钮，系统会将验证码以短信的形式发送到手机，填写验证码，最后点击“登录”按钮即可。

1.1.4 登录之后，如果不退出登录，下次进入公众号时，就无需再重新登录。

1.2 退出登录

1.2.1 点击“微云打”服务号下方菜单栏中“我的门店”-“系统设置”选项，进入“系统设置”页面。

1.2.2 点击“退出登录”按钮，即可退出此账号。

1.3 更换绑定电话

1.3.1 点击“微云打”服务号下方菜单栏中“我的门店”-“系统设置”选项，进入“系统设置”页面。

1.3.2 点击“更换绑定电话”按钮，填写原手机号和新手机号，点击“获取验证码”按钮，系统会向新手机号发送验证码，填写验证码，点击“更换”按钮即可。

2
平台管理
2.1 平台授权

商户可以通过“微云打”服务号，对已入驻的外卖平台（美团、饿了么、百度外卖）的订单进行集中管理和打印。

2.1.1 点击“微云打”服务号公下方菜单栏中“我的门店”-“平台管理”选项，进入“平台管理”页面。

2.1.2 选择需要接入的外卖平台：点击“>”按钮，进入授权页面，填写该外卖平台的账号和密码，并点击授权，即可接入平台。

2.1.2 各外卖平台的接入流程均相同。

2.2 取消授权

在“平台管理”页面，点击已授权平台右侧的“>”按钮，进入平台详情页面，点击“取消授权”按钮。

3
打印机管理
3.1 新增打印机

3.1.1 点击“微云打”服务号下方菜单栏中“我的门店”-“打印机管理”，进入“打印机管理”页面。

3.1.2 绑定打印机，目前支持两种方式绑定。

①　扫码绑定：点击“新增打印机”-“扫码绑定”，扫描微云打打印机底部标签的二维码，绑定此台打印机。

②　录入“制造编号”绑定：点击“新增打印机”-“录入制造编号绑定”，用户可通过输入打印机的制造编号和校验码（每台打印机均有单独的制造编号和校验码），完成绑定。

3.2 打印机设置

成功绑定打印机后，点击弹窗的“前去设置”按钮，进入“打印机设置”页面，即可对该打印机进行设置；

如需修改打印机的设置，在“打印机管理”页面的打印机列表中找到该打印机，点击打印机状态中的“>”按钮，进入打印机设置页面，更改设置；

还未设置过的打印机，右方会显示“待设置”状态，点击右侧“>”按钮，即可对该打印机进行设置。

3.2.1 打印机名称设置

在打印机设置页面，点击‘名称’字段的“>”，可对该打印机重新命名，否则将显示该打印机的默认名称。

3.2.2 设置打印场景

在打印机设置页面，点击“>”按钮，即可勾选该打印机的使用场景，查看打印模板；共有3种打印场景可供选择，根据需要选择其中一种或者多种打印场景。

a.外卖小票：供外卖派送时查看使用

b.订单小票：供财务对账使用

c.厨房小票：供厨师安排任务使用

d.取消小票：用户取消订单的小票

3.2.3 设置打印份数

通过点击“-”和“+”号，增加或者减少份数（每种小票最多能打印5份）。点击“保存”按钮，即可设置成功。

3.2.4 更改使用状态

打印机使用状态默认为“正常。如需暂时停用，点击状态按钮即可暂停该打印机。再次点击按钮即可重新恢复正常，并保留之前的设置。

3.2.5 解除绑定

在打印机设置页面，点击“解除绑定”按钮，可将打印机从列表中删除。

查看门店信息

点击公众号菜单栏“我的门店”-“门店信息”，即可查看门店信息。

5
订单管理
5.1 今日订单

点击微信公众号下方菜单“订单管理-今日订单”，即可进入“今日订单”页面，查看当天的订单。

有三种订单状态：

进行中：当天成功接单后尚处于进行中的订单。

已完成：当天已完成的订单。

已取消：当天已取消的订单。

5.1.1 订单详情

点击订单号右方的“>”按钮，即可查看相应的订单详情。

5.1.2 小票打印状态

每笔订单的小票打印状态有3种：

待打印：已经接单，正在准备打印小票。

已打印：已经打印过小票。（如需重新打印小票，可点击“重打”按钮。）

打印失败：打印失败和长时间未打印的订单。点击“补打”按钮，即可再次打印小票。

5.1.3 重打/补打

重打：

点击订单右侧“ ”按钮，勾选需要打印的场景（默认全部勾选，点击勾选框可取消），即可重新打印相应的小票。

补打：

点击订单右侧“ ”按钮，勾选需要打印的场景（默认全部勾选，点击勾选框可取消），即可补充打印相应的小票。

5.2 预订单

点击微信公众号下方菜单“订单管理-预订单”，进入“预订单”页面，可查看最近7天进行中的订单，以便商户能按时间的先后顺序及时准备。

PS:消费者预定好送达时间的订单即为预订单，带有“ ”标记。

5.3 订单补打

点击微信公众号下方菜单“订单管理-订单补打”，可迅速找到所有需要补打小票的订单。

5.4 订单查询

点击微信公众号下方菜单“订单管理-订单查询”，查询最近7日的订单。

（1）在输入框中输入需要查询的订单号/手机号/平台订单序号。

（2）选择需要查询的平台，订单状态和打印状态。

（3）点击搜索按钮即可查询。

6
数据报表
6.1 业绩日报

根据商户所授权的外卖平台营业状况，从订单，菜品，客户三个维度，生成的详细可视化数据报表（当天数据次日生成）。

默认显示前一天的订单日报，可查看单日订单数据，共有6项数据：

a.订单总量，即所选日期已完成和已取消的订单数量；

b.取消订单数，即所选日期已取消订单的数量；

c.订单日环比，即所选日期与前一天的订单总数相对比的增减率；

d.订单总额，即所选日期全天的已完成和已取消的订单总额；

e.取消金额，即所选日期全天取消订单的总额；

f.订单总额日环比，即所选日期与前一天的订单收入总额相比的增减率。

1)查看单日订单数据

点击“业绩日报”右侧的按钮，选择要查看的日期，即可查看详细数据。折线图可以向左滑动，直观显示不同平台营业状况的对比，帮助商家优化资源配置。

2)查看多日订单数据

点击“平台统计”右侧的图标，即可选择需要查看的起止时间（商户自行选择日期范围，区间最长为6个月）。

“微云打”可查看订单总量数据折线图，横坐标表示时间，纵坐标表示订单总量，不同平台会以不同颜色加以区分，通过选择平台图标，选择查看外卖平台订单数据的范围，直观展示不同平台连续多天的经营情况。

6.2 菜品数据报表

可统计菜品销量排名，以及在每个外卖平台的销量。

点击“菜品销量”右侧的三角形图标，报表数据可按由多到少或由少到多的顺序排列。

向右滑动屏幕，即可查看更多菜品的相关数据。

6.3 客户数据报表

选择日期后，可查看所有外卖客户的人数总量。
